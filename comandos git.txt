- git init 
crea un repositorio 

- touch [nombre.extension] crea archivo.

- git log (podes mirar los commit)

- git checkout -b [nombre de la rama] (crea una nueva rama)

- git checkout [nombre de la rama]

unir las ramas

git merge [nombre de la otra rama]

Termino de hacer el proyecto
Pasos para crear una rama:

1- desde el master hago
- git checkout -b [nombre de la rama] (crea una nueva rama)

2- cargo la rama al repositorio remoto  
git push --set-upstream origin [nombre de la rama]

Clonar una sola rama de un repositorio
git clone [url del master] -b [nombre de la rama] -- single-branch
git clone https://github.com/emiirg93/BatallaNaval_ITR.git -b Task_1 --single-branch

eliminar una rama remota:
git push origin --delete [nombre de la rama]

eliminar una rama local:
git branch -d [nombre de la rama]
